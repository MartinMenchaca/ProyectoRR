# Movilidad Inteligente POC

Prueba de concepto para la materia Sistemas Distribuidos.

El objetivo es demostrar comunicaciones distribuidas en un sistema de movilidad inteligente:

- Simulador IoT publicando ubicaciones por MQTT.
- Broker MQTT recibiendo eventos.
- Backend Node.js suscrito al broker.
- Persistencia en SQLite.
- Frontend React consultando por REST.
- Actualización en tiempo real mediante WebSocket.
- Interfaz visual tipo app moderna de movilidad.

Esta prueba no implementa todo el sistema real, solo lo necesario para demostrar las comunicaciones planteadas.
