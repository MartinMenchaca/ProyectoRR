# Documentacion UML

Esta carpeta contiene la documentacion tecnica de la POC.

- `component-diagram.puml`: componentes logicos del sistema.
- `deployment-diagram.puml`: nodos/procesos desplegados para la demostracion.

En la Fase 1 se deja la estructura base. En fases posteriores se completaran los diagramas con MQTT, backend, simulador, frontend, WebSocket y SQLite.
